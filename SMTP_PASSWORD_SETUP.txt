# WICHTIG: SMTP-Passwort einrichten

## So aktivieren Sie den E-Mail-Versand:

1. **Öffnen Sie die .env Datei:**
   ```bash
   nano /app/backend/.env
   ```

2. **Tragen Sie das SMTP-Passwort ein:**
   Suchen Sie die Zeile:
   ```
   SMTP_PASSWORD=
   ```
   
   Ändern Sie zu:
   ```
   SMTP_PASSWORD=IHR_IONOS_PASSWORT
   ```

3. **Speichern Sie die Datei:**
   - Drücken Sie `Ctrl + X`
   - Bestätigen Sie mit `Y`
   - Drücken Sie `Enter`

4. **Starten Sie das Backend neu:**
   ```bash
   sudo supervisorctl restart backend
   ```

5. **Testen Sie die Konfiguration:**
   Erstellen Sie einen Test-Lead über den Angebotsrechner unter:
   https://hamburg-paint-demo.preview.emergentagent.com/rechner
   
   Sie sollten eine E-Mail an info@ocean-maler.de erhalten.

## Aktueller Status:

✓ SMTP Host: smtp.ionos.de (Port 587, STARTTLS)
✓ SMTP User: info@ocean-maler.de
✓ From Email: info@ocean-maler.de
✓ Admin Email: info@ocean-maler.de
✗ SMTP Password: NICHT GESETZT (muss manuell eingetragen werden)

**Solange das Passwort nicht gesetzt ist, werden E-Mails nur geloggt, nicht versendet.**

## Logs überwachen:

Nach dem Eintragen des Passworts können Sie die Logs überwachen:

```bash
tail -f /var/log/supervisor/backend.out.log | grep -i "smtp\|email"
```

Erfolgreiche E-Mail-Zustellung zeigt:
```
INFO: SMTP login successful as info@ocean-maler.de
INFO: ✓ Lead notification email sent to info@ocean-maler.de
```
